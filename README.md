# 💼 LogiSoft – Projeto Final de Programação em Java

## 🧠 Sobre o Projeto

Este repositório contém o projeto final desenvolvido pela equipe **LogiSoft** como parte da disciplina de Programação do curso técnico em Informática. O sistema foi inteiramente desenvolvido em **Java** e conta com três módulos principais:

- 🔐 **Sistema de Login**
- ➕ **Calculadora Básica**
- ❌ **Jogo da Velha (Tic Tac Toe)**

O projeto tem como objetivo integrar conceitos de lógica de programação, interface gráfica, estruturação de código orientado a objetos, versionamento e organização em equipe.

---

## 🏢 Sobre a Empresa

**LogiSoft** é uma softhouse fictícia formada por estudantes, com o propósito de criar soluções tecnológicas acessíveis, criativas e funcionais. A equipe se destaca por aplicar lógica e simplicidade na criação de sistemas úteis e interativos.

### Missão  
Desenvolver, como equipe estudantil, sistemas simples e funcionais que apliquem os conhecimentos adquiridos em sala de aula, promovendo aprendizado prático por meio da lógica, da criatividade e do trabalho em grupo.

### Visão  
Consolidar o conhecimento técnico aprendido no curso através da construção de projetos úteis, educativos e interativos, mostrando que estudantes também podem criar soluções com qualidade e propósito.

### Valores
- Aprender fazendo  
- Trabalho em equipe  
- Lógica com diversão  
- Utilidade no cotidiano  
- Segurança e organização  
- Criatividade aplicada à tecnologia

## 💻 Funcionalidades

### 🔐 Sistema de Login
- Tela de autenticação com campos de usuário e senha
- Validação simples de credenciais
- Redirecionamento ao menu principal após login bem-sucedido

### ➕ Calculadora
- Operações básicas: soma, subtração, multiplicação e divisão
- Interface gráfica com botões e visor
- Validação de entradas e tratamento de erros

### ❌ Jogo da Velha
- Interface com tabuleiro 3x3
- Alternância entre jogadores X e O
- Verificação automática de vitória e empate
- Opção de reiniciar o jogo
